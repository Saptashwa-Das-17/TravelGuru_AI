## 🧭 Overview

**TravelGuru** is an AI-powered travel planning agent designed to help users create smart, personalized, and budget-friendly trips.  
It provides destination suggestions, builds itineraries, recommends transport and hotels, and responds in real time — all powered by **IBM watsonx.ai** and deployable on the **IBM Cloud Lite** tier using **Prompt Lab** and **IBM Granite foundation models**.

---

## ✨ Features

1. **Conversational AI** – Assists with trip planning, destination discovery, budgeting, and more  
2. **Free to Deploy** – Uses IBM Cloud Lite (no paid services needed)  
3. **Prompt-Engineered Agent** – Configured with a professional, travel-focused system prompt  
4. **Multi-Channel Ready** – Can be tested via Prompt Lab or deployed as a webchat or API endpoint  

---

## 🚀 Usage

You can interact with TravelGuru via:

- **IBM watsonx.ai Prompt Lab**  
- **Deployed Web Chat Widget** (if embedded)  
- **Public/Private Endpoint** (via cURL or Postman)

---

## 🧾 Quick Questions – TravelGuru

1. What is your intended travel destination?  
2. What are your preferred travel dates?  
3. How many individuals will be traveling?  
4. What is the purpose of your trip — business, leisure, or other?

---

## 💡 Example Prompts

- “Create a 4-day travel plan for Jaipur.”  
- “Suggest affordable travel options to Ooty from Bangalore.”  
- “Compare hotel prices in Delhi.”  
- “What’s a good itinerary for a solo Goa trip?”

## ⚙ How It Works
Built in IBM watsonx.ai Prompt Lab using customized system instructions

Responses generated with IBM Granite Models

No external datasets required

Agent behavior and tone defined via prompt instructions — easily adaptable for formal, luxury, or casual travel use cases

---

## ✅ Do’s and ❌ Don’ts

✅ Do:
Ask about destinations, hotels, transport, or itinerary ideas

Use conversational language

Interact via test prompt, web chat, or endpoint interface

❌ Don’t:
Share sensitive or personal information

Use it for critical decisions (medical, legal, emergency)

Expect context memory (unless connected with memory tools)
